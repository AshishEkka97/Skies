# Skies
### A simple weather app made in React Native
## Useage
#### Android
##### Install
`npm install`
##### Add API Key File
```
mkdir config
cd config
touch apiKey.js
```
##### Edit API Key File
`export const apiKey = 'YOUR_API_KEY';`
##### Run
`react native run-android`


#### iOS
##### Install
`npm install && cd ios && pod install && cd ../`
##### Add API Key File
```
mkdir config
cd config
touch apiKey.js
```
##### Edit API Key File
`export const apiKey = 'YOUR_API_KEY';`
##### Run
`react native run-ios`
